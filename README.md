## 🎮 游戏简介

"谁是卧底"是一款经典的语言推理游戏。游戏包含以下角色：

- **平民**：拿到平民词语，需要找出卧底
- **卧底**：拿到卧底词语，需要隐藏身份
- **白板**：无词语，靠观察隐藏身份（属于平民阵营）

**胜利条件：**
- **平民阵营胜利**：所有卧底被投出局
- **卧底阵营胜利**：存活平民数 ≤ 存活卧底数（⚠️ 不是平民全灭！）

## ✨ 核心特性

### 🎯 智能游戏管理
- **场/局概念**：一场游戏包含多局，连续游玩无需重复配置
- **1 小时间隔**：间隔≤1 小时自动沿用上场配置，>1 小时重新引导
- **多局统计**：自动记录每场游戏的局数、胜负历史

### 📝 标准话术模板
- **15 个标准话术**：每个步骤都有固定话术，确保体验一致性
- **游戏规则说明**：每局开始前自动说明玩法和胜利条件
- **禁止发散**：所有话术严格遵循模板，不发散、不改词

### 🔄 词语去重机制
- **1000 局不重复**：记录所有使用过的词语组合
- **智能检测**：无序检查（"奶茶/咖啡" = "咖啡/奶茶"）
- **高质量词库**：优先互联网热词、职场流行语、新兴事物

### 🎭 角色配置灵活
- **经典模式**：平民 vs 卧底
- **双卧底模式**：2 卧底 vs 平民
- **三教九流**：1 卧底 +1 白板 vs 平民
- **自定义配置**：支持任意人数和角色组合

```

     请主持人 @所有参与游戏的玩家（至少 4 人）。

### 3. 连续游戏

间隔≤1 小时，自动沿用配置：
```

     ● 已进行局数：第 1 局
     ● 参与玩家：@张三 @李四 @王五 @赵六
     ● 角色配置：平民 3 名，卧底 1 名，白板 0 名
     🎲 现在开始第 2 局，词语已重新生成！
     
     🔧 是否需要调整？
     ● 调整玩家：回复"换人"或"调整玩家"
     ● 调整配置：回复"改配置"或"换配置"
     ● 继续游戏：回复"继续"或"不调整"

## 📋 游戏规则

### 游戏流程
1. **收到词语**：私聊收到词语（星号保护，防止预览暴露）
2. **轮流发言**：线下轮流描述自己的词语（不要说词语本身）
3. **投票出局**：线下投票，在群里 @ 出投出的玩家
4. **重复流程**：直到一方获胜

### 胜利条件
| 阵营 | 胜利条件 | 说明 |
|------|----------|------|
| 平民阵营（含白板） | 所有卧底被投出局 | 卧底数=0 |
| 卧底阵营 | 存活平民数 ≤ 存活卧底数 | ⚠️ 不是平民全灭！ |

### 示例
```
场景 1：2 平民 +0 卧底 +1 白板 → 平民胜（卧底全灭）
场景 2：2 平民 +2 卧底 → 卧底胜（2≤2）
场景 3：1 平民 +2 卧底 → 卧底胜（1≤2）
场景 4：3 平民 +1 卧底 → 游戏继续（3>1）
```

## ❓ 常见问题

### Q: 如何确保词语不重复？
A: 每次生成词语前会检查 `WORD_HISTORY.json`，确保至少 1000 局内不重复。

### Q: 白板是什么？
A: 白板没有词语，靠观察他人发言隐藏身份。白板属于平民阵营，胜利条件与平民相同。

### Q: 为什么卧底胜利条件是"存活平民数 ≤ 存活卧底数"？
A: 这是标准规则。当卧底人数达到或超过平民人数时，卧底已经占据优势，无需等到平民全灭。

### Q: 如何开始新的一场游戏？
A: 距离上一局结束超过 1 小时后，再次触发游戏会自动开始新的一场，需要重新@玩家。
